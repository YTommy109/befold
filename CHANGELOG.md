# Changelog

stable リリースのユーザー影響のある変更を記録する。
dev リリースの全変更履歴は [GitHub Releases](https://github.com/YTommy109/befold/releases) を参照。
v1.5.0 より前の履歴は GitHub Releases の各タグページを参照。

## v1.6.0

### 新機能
- 戻る・進むボタンと行番号トグルをツールバーへ統合する (#181)
- 本文テキスト中のパス文字列をリンク化しクリック動作を統一する (#179)
- リンククリック挙動を整理し HTML 直接プレビューのリンク対応を追加する (#177)
- スクロール位置を UserDefaults に永続化する (#172)
- ソース表示トグルボタンをプレビュー/ソース切替のセグメントコントロールに変更する (#169)

### 修正
- セキュリティレビュー指摘事項を修正する（アップデートフローのコード署名検証、XSS 緩和ほか）(#178)
- キーボードスクロールを Safari に合わせて整理する (#176)
- 検索窓のスクロール奪取・トグル配置・表示の使い勝手を改善する (#175)
- フォルダー選択後にアプリ切替すると選択が消える問題を修正する (#167)
- プレビューのスクロール位置・表示モードが復元されない問題を修正する (#166)

## v1.5.0

### 新機能
- プレビュー内検索(Cmd+F)を追加する (#163)
- プレビューのスクロールをページ/半ページ/行単位に整理する (#158)

### 修正
- サイドバー再表示手段を左端ハンドルからツールバーボタンに統一する (#160)
- サイドバー表示切替のショートカットを Cmd+B から Cmd+S に変更する (#156)
