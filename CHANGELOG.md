# Changelog

stable リリースのユーザー影響のある変更を記録する。
dev リリースの全変更履歴は [GitHub Releases](https://github.com/YTommy109/befold/releases) を参照。
v1.5.0 より前の履歴は GitHub Releases の各タグページを参照。

## v1.7.2

### 修正
- HTML 表示直後の切替で空白表示になる問題を修正
- 完全一致・末尾改行なしのファイルで偽のトランケーション表示が出る問題を修正
- markdown-it の HTML サニタイザにあった XSS バイパスを DOMPurify で解消
- postMessage ブリッジをゲートし CSP から unsafe-inline を削除（セキュリティ強化）
- 大きなファイル・ディレクトリ表示時の一時的なフリーズを解消（ディレクトリ列挙・画像埋め込み・エンコーディング判定・mermaid読み込みをメインスレッド外／遅延化）

## v1.7.1

### 修正
- 大きめの mermaid 図で最大文字数エラーになる問題を修正
- CSV ファイルの正規フィールドが誤ったクォート解除処理で破壊される問題を修正
- 対のないクォートを含む CSV でチャンクサイズが異常化する問題を修正
- 特定条件でクラッシュする問題を修正（文字境界外アクセス）
- チャンク強制分割で UTF-8 境界・クォート状態が壊れ文字化けする問題を修正
- 巨大ファイルでエラー表示の種別が誤って丸められる問題を修正
- エンコーディング判定で NUL バイトを誤判定する問題を修正
- 読込エラー後に再描画してもエラーバナーのボタンが無効なままになる問題を修正
- 読込エラー時に表示される幻の空行と、完全読込の誤表示を修正
- チャンク境界の末尾改行が失われ行が連結される問題を修正
- 画像ズーム中にウィンドウをリサイズするとズームが打ち消される問題を修正
- 同一内容のままリネーム/別ファイルを開いた際にファイル種別表示が更新されない問題を修正
- 別ファイル切替直後に表示が不整合な中間状態になる問題を修正
- ラスター画像の初期表示をウィンドウにフィットするよう修正
- 検索バー起動時に全体を構築してフリーズする問題を修正し、表示範囲内検索に変更
- 巨大 SJIS CSV の読込を高速化し、読込中インジケータを追加
- 8KB 超のプレフィックスでエンコーディング判定に失敗した場合に全データで再判定するよう修正

## v1.7.0

## v1.6.0

- リンククリック挙動を整理し HTML 直接プレビューのリンク対応を追加する (#177)

### 修正
- セキュリティレビュー指摘事項を修正する（アップデートフローのコード署名検証、XSS 緩和ほか）(#178)
- キーボードスクロールを Safari に合わせて整理する (#176)
- 検索窓のスクロール奪取・トグル配置・表示の使い勝手を改善する (#175)
- フォルダー選択後にアプリ切替すると選択が消える問題を修正する (#167)
- プレビューのスクロール位置・表示モードが復元されない問題を修正する (#166)
