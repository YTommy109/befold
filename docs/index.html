インストール

  1. GitHub Releases から befold-vX.Y.Z.dmg をダウンロード
  2. DMG を開き、befold.app を /Applications にコピー
  3. ターミナルで以下を実行してから起動:
xattr -dr com.apple.quarantine /Applications/befold.app
配布している DMG はコード署名・公証を行っていないため、そのまま開こうとすると Gatekeeper にブロックされます。上記コマンドで quarantine 属性を除去すると起動できます。一度起動すれば、アプリ内アップデートではこの操作は不要です。

Installation

  1. Download befold-vX.Y.Z.dmg from GitHub Releases
  2. Open the DMG and copy befold.app to /Applications
  3. Run the following command in Terminal before launching:
xattr -dr com.apple.quarantine /Applications/befold.app
The distributed DMG is not code-signed or notarized, so macOS Gatekeeper will block it from opening. The command above removes the quarantine attribute. Once launched, in-app updates don't require this step again.
